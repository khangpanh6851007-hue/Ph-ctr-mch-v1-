# 🐾 Phước Cô Thảo - Website Tìm Kiếm & Chuộc Thú Cưng

Website giới thiệu dịch vụ tìm kiếm, hỗ trợ định vị và chuộc lại thú cưng bị thất lạc xuyên quốc gia. Dự án được thiết kế tối ưu, gọn nhẹ, phản hồi tốt trên cả giao diện di động lẫn máy tính.

---

## 🚀 Tính Năng Chính

* **Giao diện hiện đại & Responsive:** Tối ưu trải nghiệm người dùng trên điện thoại di động và máy tính bảng.
* **Liên hệ nhanh (Call-to-Action):** Tích hợp nút gọi điện trực tiếp qua Hotline.
* **Tích hợp Mạng xã hội:** Liên kết tới trang cá nhân/Fanpage Facebook.
* **Định vị vị trí:** Tích hợp đường dẫn và chỉ đường qua Google Maps.

---

## 🛠️ Công Nghệ Sử Dụng

* **HTML5:** Cấu trúc trang web chuẩn hóa.
* **CSS3:** Thiết kế giao diện Flexbox, Grid Layout, Responsive Web Design.
